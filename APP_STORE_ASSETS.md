# App Store Assets Guide

## Required Screenshots

### iPhone 6.7" (1290 x 2796 pixels)

1. **Feed Screen** - Show professional networking feed with posts
2. **Network Screen** - Display connections and suggestions
3. **Profile Screen** - Highlight professional profile features
4. **Discover Screen** - Show music discovery
5. **Post Creation** - Demonstrate post creation with media

### iPhone 6.5" (1242 x 2688 pixels)

Same as 6.7" but resized

### iPad Pro 12.9" (2048 x 2732 pixels)

1. **Dashboard View** - Show tablet-optimized layout
2. **Split View** - Demonstrate multitasking
3. **Network Management** - Full tablet experience

## App Icon Requirements

### iOS

- 1024x1024px PNG (no transparency)
- No rounded corners (iOS adds automatically)
- No text in icon

### Android

- Adaptive icon: 1024x1024px foreground + background
- Legacy icon: 512x512px PNG

## Promotional Graphics

### Feature Graphic (Android)

- 1024 x 500 pixels
- Highlight key features
- Include app icon and branding

### Video Preview

- 15-30 seconds
- Show app in action
- Highlight unique features
- Include captions
- Export in 1080p or 4K
